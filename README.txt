Программа: xml_valid
Служит для проверки файлов xml по xsd и последующей трансформации по xslt шаблону.
Если входящий или преобразованный файл не будет соответствовать xsd программа не сохранит результирующий файл xml

--------------------------------------------------
ИНСТРУКЦИЯ:
--Введите имя и адрес xml в формате D://test//test1.xml
--Введите имя и адрес xsd-схемы в формате D://test//xsdschema.xsd
--Введите имя и адрес xslt шаблона в формате D://test//sh.xsl
--Введите имя и адрес результирующего xml в формате D://test//newtest.xml
